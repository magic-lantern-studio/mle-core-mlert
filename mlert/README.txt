This directory is the top of the development tree for the Magic Lantern
Runtime Engine (MLERT) SDK.

Contents:

common    - The common components shared by all targets of the MLERT SDK.
android   - The development build area for the Android target.
win32     - The development build area for the Windows target.
java      - The development build area for the Java target.
linux     - The development build area for the Linux taget.
rehearsal - The development buid area for the Reharsal Player target.
noship  - An area where non-shipable components are managed (e.g. testing).
